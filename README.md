## Описание функционала  
Данный проект представляет собой настольное приложение, разработанное с использованием технологии WPF и языка программирования C#.  
Основные функции приложения:  
- Добавление, редактирование и удаление записей (пользователи, категории, платежи и др.);  
- Отображение данных в графическом интерфейсе;  
- Навигация между окнами и использование нескольких интерфейсных страниц;  
- Работа с базой данных через EDMX-модель;  
- Возможность смены темы оформления интерфейса.  

## Структура проекта  

| Путь / Файл | Назначение |
|--------------|------------|
| **App.config** | Файл конфигурации приложения (подключение к БД, настройки) |
| **Model.edmx**, **Model.Context.cs**, **Payment.cs**, **User.cs**, **Category.cs** | Модель данных и контекст Entity Framework |
| **MainWindow.xaml / MainWindow.xaml.cs** | Главное окно приложения |
| **Dictionary.xaml / Dictionary.xaml.cs** | Тема интерфейса |
| **AnotherTheme.xaml** | Альтернативная тема интерфейса |
| **Pages** | папка с страницами проекта |
| **Папка “Документы”** | Содержит отчёты и материалы по учебной практике |

## Используемые технологии  
- **C#**  
- **.NET Framework**  
- **Windows Presentation Foundation (WPF)**  
- **Entity Framework**  
- **Visual Studio 2022**
